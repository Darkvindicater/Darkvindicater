## AutoDiag AI

AutoDiag AI is designed as a mobile-first vehicle health tool using built-in phone sensors.

- Uses microphone observations for rattles, squeals, hums, and engine sounds
- Uses accelerometer-style notes for vibration patterns
- Uses gyroscope-style notes for wobble, shake, and motion patterns
- Creates maintenance alerts and safety guidance
- Helps drivers prepare better questions before visiting a mechanic
